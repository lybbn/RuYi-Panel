# 文件管理

tags: 文件,上传,下载,编辑,权限,压缩,解压

## 文件浏览器

- 「文件」页面提供Web文件管理器
- 左侧目录树，右侧文件列表
- 显示文件大小、修改时间

## 基本操作

### 创建
- 新建文件：右键菜单 → 「新建文件」
- 新建目录：右键菜单 → 「新建目录」

### 编辑
- 双击文件打开在线编辑器
- 支持语法高亮
- 修改后保存

### 上传下载
- 拖拽上传：直接拖文件到文件管理器
- 按钮上传：点击「上传」选择文件
- 下载：右键文件 → 「下载」

### 压缩解压
- 右键文件/目录 → 「压缩」
- 支持格式：zip、tar.gz
- 右键压缩包 → 「解压」

### 权限管理
- 右键文件 → 「权限」
- 可设置读/写/执行权限
- 可修改文件所有者

### 批量操作
- 支持批量复制、移动、删除、压缩
- 勾选多个文件/目录后选择批量操作

## 回收站

- 删除的文件默认进入回收站（可在设置中开关）
- 回收站中可恢复已删除文件
- 可清空回收站永久删除
- 支持批量恢复和批量永久删除

## 安全限制

- 系统关键目录禁止删除（如C盘、Windows目录、Linux系统目录）
- 如意面板安装目录禁止删除
- 黑名单端口禁止绑定（21、25、3306、6789）
