# 文件管理

tags: 文件,上传,下载,编辑,权限,压缩,解压,回收站,批量操作

## 文件浏览器

- 「文件」页面提供Web文件管理器
- 左侧目录树，右侧文件列表
- 显示文件大小、修改时间、文件类型
- 支持面包屑导航

## 基本操作

### 创建
- 新建文件：右键菜单 → 「新建文件」
- 新建目录：右键菜单 → 「新建目录」
- 支持设置文件编码

### 编辑
- 双击文件打开在线编辑器
- 支持语法高亮（HTML/CSS/JS/Python/PHP等）
- 支持代码折叠
- 支持查找替换
- 修改后保存

### 上传下载
- 拖拽上传：直接拖文件到文件管理器
- 按钮上传：点击「上传」选择文件
- 支持多文件上传
- 支持文件夹上传
- 下载：右键文件 → 「下载」
- 支持批量下载（自动打包为zip）

### 压缩解压
- 右键文件/目录 → 「压缩」
- 支持格式：zip、tar.gz、7z
- 右键压缩包 → 「解压」
- 支持解压到指定目录
- 支持密码解压

### 权限管理
- 右键文件 → 「权限」
- 可设置读/写/执行权限
- 可修改文件所有者
- 可修改文件所属组
- 支持递归修改权限

### 批量操作
- 支持批量复制、移动、删除、压缩
- 勾选多个文件/目录后选择批量操作
- 支持全选/反选

## 文件搜索

### 搜索功能
- 支持按文件名搜索
- 支持按文件内容搜索
- 支持正则表达式搜索
- 支持指定目录搜索

### 搜索选项
- 区分大小写
- 全字匹配
- 包含子目录
- 文件类型过滤

## 文件预览

### 图片预览
- 支持常见图片格式预览（JPG/PNG/GIF/WebP）
- 支持图片缩放
- 支持图片信息查看

### 文本预览
- 支持文本文件快速预览
- 支持代码文件语法高亮预览
- 支持Markdown预览

### 视频音频
- 支持常见视频格式播放
- 支持常见音频格式播放

## 回收站

- 删除的文件默认进入回收站（可在设置中开关）
- 回收站中可恢复已删除文件
- 可清空回收站永久删除
- 支持批量恢复和批量永久删除
- 支持回收站自动清理

## 文件分享

### 分享链接
- 支持生成文件分享链接
- 支持设置分享密码
- 支持设置分享有效期
- 支持设置下载次数限制

## 安全限制

- 系统关键目录禁止删除（如C盘、Windows目录、Linux系统目录）
- 如意面板安装目录禁止删除
- 黑名单端口禁止绑定（21、25、3306、6789）
- 敏感文件访问限制

## 文件传输

### FTP支持
- 支持FTP文件传输
- 配置FTP服务器
- 管理FTP用户

### SFTP支持
- 支持SFTP安全传输
- 使用SSH密钥认证

## 常用目录

### 快捷访问
- 网站根目录
- 面板安装目录
- 备份目录
- 日志目录
- 临时目录

### 书签功能
- 支持添加常用目录书签
- 快速跳转到书签目录

## 使用技巧

### 快捷键
- Ctrl+C：复制
- Ctrl+V：粘贴
- Ctrl+X：剪切
- Delete：删除
- F2：重命名
- F5：刷新

### 右键菜单
- 打开：打开文件或目录
- 编辑：编辑文件
- 下载：下载文件
- 复制：复制文件
- 移动：移动文件
- 删除：删除文件
- 重命名：重命名文件
- 权限：修改权限
- 压缩：压缩文件
- 属性：查看文件属性

## 注意事项

- 大文件上传可能需要较长时间
- 编辑大文件可能影响性能
- 删除操作请谨慎，重要文件建议备份
- 权限修改可能影响网站访问
- 压缩大文件可能需要较长时间
