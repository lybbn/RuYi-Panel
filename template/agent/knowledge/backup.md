# 备份与恢复

tags: 备份,恢复,网站备份,数据库备份,云存储备份,面板备份,备份编排,定时备份,加密备份

## 备份概述

如意面板提供全面的备份功能，支持网站、数据库、目录和面板配置的备份，支持本地和云存储备份。

## 网站备份

### 手动备份
1. 「网站」→ 选择网站 →「备份」
2. 点击「创建备份」
3. 备份文件存储在备份目录中
4. 支持备份网站文件和配置

### 恢复网站
1. 网站详情 →「备份」列表
2. 选择要恢复的备份点
3. 点击「恢复」
4. 恢复完成后自动重启网站

### 备份内容
- 网站根目录文件
- Nginx配置文件
- SSL证书文件
- 网站配置信息

## 数据库备份

### 手动备份
1. 「数据库」→ 选择数据库 →「备份」
2. 点击「创建备份」
3. 支持MySQL、PostgreSQL、MongoDB、Redis备份

### 恢复数据库
1. 数据库列表 →「备份」
2. 选择备份文件
3. 点击「恢复」
4. 恢复完成后自动重启数据库

### 备份格式
- MySQL：SQL文件（.sql）
- PostgreSQL：SQL文件（.sql）
- MongoDB：JSON文件
- Redis：RDB文件

## 目录备份

### 备份目录
1. 「备份」→「目录备份」
2. 选择要备份的目录
3. 设置备份排除规则
4. 创建备份

### 备份排除
- 支持排除指定目录
- 支持排除指定文件类型
- 支持排除大于指定大小的文件

## 面板备份

### 面板配置备份
- 「备份」→「面板备份」页面
- 备份面板所有配置数据
- 支持选择备份内容：
  - 网站配置
  - 数据库数据
  - 计划任务
  - SSL证书
  - WAF配置
  - 告警配置
  - 节点配置
  - 面板设置

### 备份操作
1. 点击「创建备份」
2. 输入备份名称
3. 选择备份数据类型
4. 选择存储位置（本地/云存储）
5. 配置备份排除规则
6. 开始备份

### 面板恢复
- 选择备份文件
- 选择恢复内容
- 开始恢复
- 恢复完成后自动重启

## 定时备份

### 计划任务备份
- 在「计划任务」中创建备份任务
- 支持备份网站、数据库、目录
- 设置执行周期（每天/每周/每月）
- 自动执行备份

### 备份策略建议
- **每日备份**：重要网站和数据库
- **每周备份**：一般网站和配置
- **每月备份**：完整系统备份

## 云存储备份

### 备份到云端
- 支持备份到多种云存储
- 阿里云OSS、腾讯云COS、华为云OBS等
- 支持OneDrive、WebDAV
- 支持自建MinIO

### 配置云备份
1. 先在「云存储」中添加账号
2. 创建备份时选择云存储账号
3. 设置云端备份路径
4. 支持备份加密

### 云端备份管理
- 查看云端备份文件
- 下载云端备份
- 从云端恢复
- 删除云端备份

## 备份加密

### 加密备份
- 支持备份文件加密
- 使用AES加密算法
- 设置加密密码
- 恢复时需要输入密码

### 加密建议
- 重要数据建议加密备份
- 加密密码请妥善保管
- 建议使用强密码

## 备份编排

### 编排备份
- 「备份」→「备份编排」页面
- 创建备份编排任务
- 组合多种备份类型
- 设置备份策略

### 编排选项
- 选择备份目标（网站/数据库/目录）
- 选择存储位置
- 设置保留策略
- 配置通知选项

## 备份存储

### 本地存储
- 备份文件存储在本地备份目录
- 可自定义备份目录路径
- 支持备份文件管理

### 存储管理
- 查看备份文件列表
- 查看备份文件大小
- 下载备份文件
- 删除备份文件
- 上传备份文件

## 使用场景

### 数据保护
- 定期备份重要数据
- 多地备份提高安全性
- 加密备份保护隐私

### 灾难恢复
- 服务器故障快速恢复
- 数据误删快速恢复
- 系统迁移数据恢复

### 版本管理
- 保留多个备份版本
- 按时间点恢复数据
- 对比不同版本差异

## 注意事项

- 备份文件会占用存储空间，建议定期清理
- 重要备份建议多地存储
- 定期验证备份文件可用性
- 恢复操作前建议先备份当前数据
- 加密备份密码丢失将无法恢复
