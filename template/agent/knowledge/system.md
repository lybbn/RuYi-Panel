# 系统管理

## 系统信息

- 首页仪表盘展示服务器概览
- CPU使用率、内存使用率、磁盘使用率
- 系统运行时间、操作系统版本
- 网络流量监控（上传/下载速率）
- 负载均衡（1分钟/5分钟/15分钟平均负载）
- 进程数量统计

## 防火墙

### 端口管理
- 「安全」→「防火墙」页面
- 查看已开放端口列表
- 添加放行规则：指定端口号 + 协议（TCP/UDP）
- 删除规则即关闭对应端口
- 支持端口范围放行（如8000-9000）

### 端口转发
- 支持端口转发规则配置
- 将外部端口流量转发到内部IP:端口
- 适用于内网服务暴露、负载均衡等场景

## SSH管理

### SSH配置
- 「安全」→「SSH」页面
- 修改SSH端口（默认22）
- 查看SSH配置信息
- 支持密钥认证配置

### Fail2Ban防暴力破解
- 通过应用商店安装Fail2Ban（仅Linux）
- 保护SSH等服务免受暴力破解
- 可查看被封禁的IP列表
- 可手动解封IP
- 可配置监控规则和封禁策略

## 监控

### 系统监控
- 「监控」页面展示历史监控数据
- CPU、内存、磁盘IO、网络IO趋势图
- 系统负载监控
- 支持自定义监控时间范围
- 支持监控数据采集间隔配置（默认5分钟）
- 可开启/关闭监控功能

### 进程管理
- 「安全」→「进程管理」页面
- 查看系统所有运行进程
- 显示进程PID、CPU占用、内存占用、启动命令
- 支持终止指定进程
- 支持按进程名、PID筛选

### 网络连接
- 「安全」→「网络连接」页面
- 查看当前网络连接状态
- 显示本地地址、远程地址、连接状态、进程PID
- 支持按协议（TCP/UDP）筛选
- 识别异常连接和可疑进程

## 计划任务

### 添加任务
1. 「计划任务」页面 →「添加任务」
2. 选择任务类型：
   - Shell脚本：执行自定义命令或脚本
   - 备份网站：自动备份指定网站
   - 备份数据库：自动备份指定数据库
   - 备份目录：备份指定目录
   - 访问URL：定时访问指定网址
   - AI任务：定时执行AI智能任务
3. 设置执行周期（Cron表达式或可视化选择）
4. 填写任务内容
5. 保存后按周期自动执行

### 任务管理
- 查看任务执行历史和日志
- 手动执行任务
- 启用/禁用任务
- 支持任务分组管理

### AI定时任务
- 支持定时执行AI智能任务
- 可配置AI提示词（prompt）
- 支持设置AI任务上下文来源
- 支持静默模式执行
- 可查看AI任务执行结果

## 面板升级

### 在线升级
- 页面头部 更多-》检查更新-》确认是否升级
- 有新版本时会显示更新提示，点击即可在线升级

### 离线升级
适用场景：服务器无法访问外网、内网环境、需要指定版本升级

#### 操作步骤
1. 找如意面板RuYi-Panel作者lybbn，询问最新如意面板离线升级包（zip格式，文件名如 ruyi.zip）
2. 登录如意面板，进入「面板设置」→「关于面板」页面
3. 找到「离线升级」区域，将升级包拖拽到上传区域或点击选择文件
4. 上传完成后点击「开始升级」按钮
5. 系统会弹出日志窗口，实时显示升级进度
6. 升级完成后面板会自动重启

#### 升级包要求
- 格式：zip压缩包
- 大小限制：不超过100MB
- 包内结构：需包含 ruyi/ 目录及完整的面板文件

#### 升级机制说明
- 升级前自动备份旧版文件到 data/backup/panel/ 目录
- 校验升级包版本号，不允许降级操作
- 替换面板文件后自动执行数据库迁移和初始化脚本
- 升级失败时可从备份目录恢复

### 升级日志
- 升级过程中可查看实时日志
- 日志文件位于 data/logs/ruyi_updatepanel.log
- 升级失败时请查看日志排查问题

## 面板命令行工具

Linux系统下可使用 `ruyi-cmd` 命令行工具管理面板：

```bash
ruyi-cmd status      # 查看运行状态
ruyi-cmd start       # 启动面板服务
ruyi-cmd stop        # 停止面板服务
ruyi-cmd restart     # 重启面板服务
ruyi-cmd version     # 查看版本信息
ruyi-cmd repair      # 修复如意面板
ruyi-cmd setport     # 修改面板端口
ruyi-cmd setsafepath # 修改面板安全入口
ruyi-cmd setuser     # 修改面板用户名
ruyi-cmd setpass     # 修改面板密码
ruyi-cmd disableotp  # 关闭OTP两步验证
ruyi-cmd clearcache  # 清理面板缓存
```
