# 日志管理

tags: 日志,操作日志,访问日志,错误日志,慢查询日志,系统日志,审计

## 日志概述

如意面板提供日志管理功能，记录系统操作和访问信息，便于审计和问题排查。

## 操作日志

### 操作日志记录
- 「日志」→「操作日志」页面
- 记录所有面板操作
- 包含操作用户、操作模块、操作内容、IP地址、时间

### 日志模块
- **用户登录**：登录/登出记录
- **网站管理**：网站创建/修改/删除操作
- **数据库管理**：数据库操作记录
- **文件管理**：文件操作记录
- **软件管理**：应用安装/卸载操作
- **容器管理**：Docker操作记录
- **安全操作**：防火墙/SSH操作
- **面板设置**：配置修改记录
- **计划任务**：定时任务操作
- **WAF防护**：WAF配置变更
- **系统监控**：监控配置变更
- **执行命令**：命令执行记录

### 日志查询
- 按时间范围筛选
- 按操作模块筛选
- 按操作用户筛选
- 按操作内容搜索

### 日志清理
- 支持手动清空日志
- 支持按时间范围清理
- 自动清理过期日志

## 网站日志

### 访问日志
- 网站详情 → 「日志」→「访问日志」
- 记录所有HTTP访问请求
- 包含客户端IP、请求URL、状态码、响应时间
- 支持日志下载

### 错误日志
- 网站详情 → 「日志」→「错误日志」
- 记录网站错误信息
- 包含错误类型、错误详情、发生时间
- 便于问题排查

### 日志配置
- 支持开启/关闭访问日志
- 支持开启/关闭错误日志
- 支持日志格式配置

## 数据库日志

### MySQL日志

#### 错误日志
- MySQL应用 → 「日志」→「错误日志」
- 记录MySQL启动、运行、停止过程中的错误
- 便于排查数据库问题

#### 慢查询日志
- MySQL应用 → 「日志」→「慢查询日志」
- 记录执行时间超过阈值的SQL语句
- 可配置慢查询阈值（默认10秒）
- 便于SQL性能优化

#### 通用查询日志
- 记录所有SQL语句
- 用于调试和审计
- 生产环境建议关闭

### Redis日志
- Redis应用 → 「日志」
- 记录Redis运行日志
- 包含连接、持久化、错误信息

## 系统日志

### 系统日志查看
- 支持查看系统日志
- 包含系统启动、服务运行、错误信息
- 支持按时间筛选

### 登录日志
- 「安全」→「登录历史」
- 记录所有登录尝试
- 包含成功和失败的登录
- 显示登录IP、时间、浏览器

## WAF日志

### 攻击日志
- 「WAF」→「攻击日志」
- 记录所有被WAF拦截的请求
- 包含攻击类型、来源IP、请求URL
- 支持按类型、IP、时间筛选

### 日志分析
- 攻击趋势分析
- 攻击类型统计
- 攻击来源分析
- Top攻击IP排行

## Nginx日志

### Nginx访问日志
- Nginx应用 → 「日志」→「访问日志」
- 记录所有HTTP请求
- 支持自定义日志格式

### Nginx错误日志
- Nginx应用 → 「日志」→「错误日志」
- 记录Nginx运行错误
- 包含配置错误、连接错误

## Docker日志

### 容器日志
- 容器详情 → 「日志」
- 实时查看容器标准输出
- 支持日志下载
- 支持日志搜索

### 日志配置
- 支持配置日志驱动
- 支持配置日志轮转
- 支持配置日志大小限制

## 日志存储

### 存储位置
- 操作日志：SQLite数据库
- 网站日志：Nginx logs目录
- 数据库日志：各数据库日志目录
- 系统日志：系统日志目录

### 日志轮转
- 支持日志轮转配置
- 自动清理过期日志
- 支持日志压缩

## 使用场景

### 问题排查
- 通过错误日志定位问题
- 通过访问日志分析流量
- 通过操作日志追溯变更

### 安全审计
- 审计用户操作
- 分析攻击来源
- 合规性检查

### 性能优化
- 分析慢查询日志
- 优化SQL语句
- 分析访问模式

## 注意事项

- 日志会占用磁盘空间，建议定期清理
- 生产环境建议关闭调试日志
- 敏感信息注意脱敏处理
- 重要日志建议备份
