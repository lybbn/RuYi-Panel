# 云存储管理

tags: 云存储,OSS,COS,S3,WebDAV,OneDrive,挂载,备份,rclone,MinIO

## 云存储概述

如意面板支持多种云存储服务，可用于数据备份、文件存储和云盘挂载。

## 支持的云存储服务

### 国内云存储
- **阿里云 OSS**：对象存储服务
- **腾讯云 COS**：对象存储服务
- **华为云 OBS**：对象存储服务
- **百度云 BOS**：对象存储服务
- **京东云 OSS**：对象存储服务
- **金山云 KS3**：对象存储服务
- **天翼云 ZOS**：对象存储服务
- **七牛云 Kodo**：对象存储服务

### 国际云存储
- **Cloudflare R2**：S3兼容存储
- **谷歌云存储**：Google Cloud Storage

### 自建存储
- **MinIO**：S3兼容的自建对象存储
- **WebDAV**：WebDAV协议存储
- **OneDrive**：微软云盘

## 添加云存储账号

1. 「云存储」页面 →「添加账号」
2. 选择云存储服务商
3. 填写配置信息：
   - 账号名称（自定义）
   - AccessKey / SecretKey
   - Endpoint（服务端点）
   - Region（地域）
   - Bucket（存储桶）
   - 云端备份路径
4. 点击「测试连接」验证配置
5. 保存配置

### 协议支持
- **S3协议**：兼容AWS S3，支持大多数云存储
- **原生SDK**：使用云厂商官方SDK
- **WebDAV协议**：WebDAV标准协议
- **OneDrive API**：微软Graph API

## 云存储管理

### 账号管理
- 查看所有云存储账号列表
- 编辑账号配置
- 删除账号
- 设置默认备份账号
- 测试账号连接

### SDK管理
- 部分云存储需要安装SDK
- 支持一键安装所需SDK包
- 查看SDK安装状态
- 支持的SDK：boto3、oss2、cos-sdk、qiniu、minio、webdavclient3、msal、rclone

### 存储配额
- 可手动设置存储配额
- 查看已用存储空间
- 配额超限告警

## 云盘挂载

### 挂载功能
- 「云存储」→「挂载」页面
- 将云存储挂载为本地目录
- 支持开机自动挂载
- 支持自定义挂载选项

### 挂载操作
1. 选择已配置的云存储账号
2. 设置本地挂载路径
3. 配置挂载选项
4. 点击挂载
5. 挂载成功后可像本地目录一样操作

### rclone支持
- 使用rclone挂载工具
- 支持FUSE挂载
- 支持多种云存储协议

## 云存储备份

### 备份到云端
- 面板备份支持选择云存储账号
- 网站备份、数据库备份支持云存储
- 自动备份任务支持云存储
- 支持备份文件加密

### 备份管理
- 查看云端备份文件列表
- 下载云端备份文件
- 从云端备份恢复
- 删除云端备份文件

## 使用场景

### 数据备份
- 定期备份网站和数据库到云端
- 多地域备份提高数据安全性
- 备份文件异地容灾

### 文件存储
- 大文件存储到云端
- 静态资源CDN加速
- 文件共享和分发

### 云盘使用
- 挂载云盘作为本地存储
- 扩展服务器存储空间
- 多服务器共享存储

## 注意事项

- AccessKey请妥善保管，不要泄露
- 建议使用子账号并限制权限
- 挂载功能依赖rclone，请先安装
- 大文件传输注意网络带宽
- 建议开启备份加密保护敏感数据
