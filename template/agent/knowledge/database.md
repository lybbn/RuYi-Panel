# 数据库管理

tags: 数据库,MySQL,Redis,phpMyAdmin,备份,恢复,远程数据库

## MySQL管理

### 创建数据库
1. 进入「数据库」页面，点击「添加数据库」
2. 填写数据库名、用户名、密码
3. 选择访问权限：本地服务器 / 所有人 / 指定IP
4. 点击提交创建

### 修改密码
1. 数据库列表 → 点击对应数据库的「改密」
2. 输入新密码
3. 确认后立即生效

### 删除数据库
- 数据库列表 → 点击「删除」
- 需确认后才会执行删除操作

### 远程数据库
- 支持添加远程MySQL/SqlServer/MongoDB/PgSql/Redis数据库
- 填写远程服务器地址、端口、账号密码即可连接管理

### 数据库备份与恢复
- 数据库列表 → 「备份」→「创建备份」
- 备份文件存储在备份目录中
- 支持从备份恢复数据库

### MySQL配置
- 应用商店中MySQL应用支持查看和编辑配置文件
- 支持性能参数调整
- 支持查看错误日志和慢查询日志
- 修改配置后需重启MySQL生效

### 远程访问
- 默认仅允许本地连接
- 如需远程访问：创建数据库时选择访问权限为「所有人」或「指定IP」
- 建议配合防火墙限制访问IP

## Redis管理

### Redis状态
- 应用商店中Redis应用支持查看运行状态
- 支持查看和编辑Redis配置文件

### Redis数据管理
- 数据库页面支持查看Redis数据库列表
- 支持查看和操作Redis Key

## phpMyAdmin

- 通过Docker广场可一键安装phpMyAdmin
- 安装时需关联MySQL服务
- 使用数据库用户名和密码登录
- 支持可视化管理表结构、数据导入导出
