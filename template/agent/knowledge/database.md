# 数据库管理

tags: 数据库,MySQL,PostgreSQL,MongoDB,Redis,SQLite,phpMyAdmin,备份,恢复,远程数据库

## MySQL管理

### 创建数据库
1. 进入「数据库」页面，点击「添加数据库」
2. 填写数据库名、用户名、密码
3. 选择访问权限：本地服务器 / 所有人 / 指定IP
4. 点击提交创建

### 修改密码
1. 数据库列表 → 点击对应数据库的「改密」
2. 输入新密码
3. 确认后立即生效

### 删除数据库
- 数据库列表 → 点击「删除」
- 需确认后才会执行删除操作

### 远程数据库
- 支持添加远程MySQL/SqlServer/MongoDB/PgSql/Redis数据库
- 填写远程服务器地址、端口、账号密码即可连接管理

### 数据库备份与恢复
- 数据库列表 → 「备份」→「创建备份」
- 备份文件存储在备份目录中
- 支持从备份恢复数据库

### MySQL配置
- 应用商店中MySQL应用支持查看和编辑配置文件
- 支持性能参数调整
- 支持查看错误日志和慢查询日志
- 修改配置后需重启MySQL生效

### 远程访问
- 默认仅允许本地连接
- 如需远程访问：创建数据库时选择访问权限为「所有人」或「指定IP」
- 建议配合防火墙限制访问IP

## PostgreSQL管理

### 创建数据库
1. 「数据库」→「PostgreSQL」页面
2. 点击「添加数据库」
3. 填写数据库名、用户名、密码
4. 选择访问权限
5. 保存

### 数据库操作
- 支持数据库备份恢复
- 支持数据库导入导出
- 支持查看数据库大小

### 配置管理
- 查看PostgreSQL配置文件
- 支持性能参数调整
- 查看PostgreSQL日志

## MongoDB管理

### 创建数据库
1. 「数据库」→「MongoDB」页面
2. 点击「添加数据库」
3. 填写数据库名、用户名、密码
4. 保存

### 数据库操作
- 查看数据库列表
- 查看集合列表
- 支持数据库备份恢复
- 支持数据导入导出

### 配置管理
- 查看MongoDB配置文件
- 支持性能参数调整
- 查看MongoDB日志

## Redis管理

### Redis状态
- 应用商店中Redis应用支持查看运行状态
- 支持查看和编辑Redis配置文件

### Redis数据管理
- 数据库页面支持查看Redis数据库列表
- 支持查看和操作Redis Key
- 支持Key的增删改查
- 支持查看Key的TTL

### Redis配置
- 查看Redis配置文件
- 支持内存限制配置
- 支持持久化配置
- 查看Redis日志

## SQLite管理

### SQLite数据库
- 支持查看SQLite数据库文件
- 支持执行SQL语句
- 支持表结构查看
- 支持数据导入导出

## phpMyAdmin

- 通过Docker广场可一键安装phpMyAdmin
- 安装时需关联MySQL服务
- 使用数据库用户名和密码登录
- 支持可视化管理表结构、数据导入导出

## 数据库优化

### MySQL优化
- 支持表优化操作
- 支持索引优化建议
- 支持查询缓存配置

### 性能监控
- 查看数据库连接数
- 查看数据库查询统计
- 查看数据库内存使用

## 数据库安全

### 访问控制
- 支持IP白名单访问
- 支持用户权限管理
- 支持密码强度检查

### 数据加密
- 支持数据传输加密
- 支持数据存储加密

## 使用场景

### 网站数据库
- 为网站创建MySQL/PostgreSQL数据库
- 配置数据库用户权限
- 定期备份网站数据库

### 缓存加速
- 使用Redis作为缓存
- 配置Redis内存限制
- 监控Redis运行状态

### 数据存储
- 使用MongoDB存储文档数据
- 使用SQLite存储轻量数据
- 使用PostgreSQL存储复杂数据

## 注意事项

- 数据库密码请使用强密码
- 生产环境建议限制数据库访问IP
- 定期备份重要数据库
- 修改数据库配置前建议备份
- 大数据库导入导出可能需要较长时间
