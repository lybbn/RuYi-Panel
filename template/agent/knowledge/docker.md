# Docker管理

tags: Docker,容器,镜像,广场,Compose,一键部署

## 容器管理

### 查看容器
- 「Docker」页面展示所有容器列表
- 显示容器名称、状态、镜像、端口映射、创建时间

### 容器操作
- **启动/停止/重启**：一键操作容器生命周期
- **查看日志**：实时查看容器标准输出日志
- **进入终端**：Web终端直接进入容器内部
- **资源限制**：可设置容器CPU和内存限制

## 镜像管理

### 镜像列表
- 展示本地所有Docker镜像
- 显示镜像名、标签、大小、创建时间
- 支持删除未使用的镜像释放空间

### 镜像仓库
- 「Docker」→「仓库」页面
- 可添加私有仓库地址和认证信息
- 添加后拉取镜像时自动使用对应仓库

## Docker广场

Docker广场提供一键部署常用应用的功能。

### 可用应用

#### 建站类
- **WordPress**：开源博客和CMS系统（需关联MySQL）
- **Discuz!**：论坛系统

#### 数据库类
- **MySQL**：关系型数据库（9.0/8.4/5.7版本）
- **PostgreSQL**：开源关系型数据库（17.4/16.8版本）

#### AI/大模型类
- **Ollama**：本地运行大语言模型
- **OpenWebUI**：Ollama的Web管理界面
- **OpenClaw**：AI编程助手
- **vLLM**：高性能LLM推理引擎
- **Hermes Agent**：开源AI智能体框架，支持多平台消息网关和技能记忆

#### 实用工具类
- **phpMyAdmin**：MySQL可视化管理
- **FRP客户端/服务端**：内网穿透工具
- **Playwright**：浏览器自动化工具

#### 运行环境类
- **RabbitMQ**：消息队列服务

#### 系统类
- **guacd**：远程桌面网关服务

### 一键部署
1. 「Docker」→「广场」页面
2. 选择需要安装的应用
3. 填写配置参数（端口、密码等）
4. 点击安装，系统自动拉取镜像并创建容器
5. 安装完成后可在列表中管理

### 应用管理
- 已安装的广场应用在列表中展示
- 支持启动、停止、卸载操作
- 部分应用支持安装多个实例

## Docker配置

- 应用商店中Docker应用支持查看和编辑Docker配置文件
- Windows下为Docker Desktop
- Linux下为Docker Engine
