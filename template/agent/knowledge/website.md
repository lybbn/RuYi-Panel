# 网站管理

tags: 网站,域名,站点,Nginx,SSL,证书,HTTPS,静态网站,Python项目,PHP项目,Go项目,Node.js项目,反向代理,负载均衡

## 创建网站

如意面板支持以下网站类型：

### 静态网站
1. 进入「网站」页面，点击「创建网站」
2. 选择「静态网站」类型
3. 填写域名（支持泛域名 *.example.com），可绑定多个域名
4. 设置根目录路径
5. 选择站点分组
6. 点击提交，系统自动生成Nginx配置并创建站点

### Python项目
1. 在「Python」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径和启动方式
4. 选择Python版本（需先在应用商店安装对应版本）
5. 配置框架类型（Django/Flask等）和启动模块
6. 系统自动创建站点并配置Nginx反向代理

### PHP项目
1. 在「PHP」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径
4. 选择PHP版本（需先在应用商店安装对应版本）
5. 系统自动创建站点并配置PHP-FPM

### Go项目
1. 在「Go」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径和启动命令
4. 选择Go版本（需先在应用商店安装对应版本）
5. 系统自动创建站点并配置Nginx反向代理

### Node.js项目
1. 在「Node.js」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径和启动命令
4. 选择Node.js版本
5. 配置包管理器（npm/yarn/pnpm）
6. 系统自动创建站点并配置Nginx反向代理

## 域名管理

- 每个网站支持绑定多个域名（含端口）
- 网站详情 → 「域名管理」→「添加域名」
- 域名格式：domain.com 或 domain.com:8080
- 域名变更后自动更新Nginx配置
- 支持泛域名绑定

## SSL证书

### Let's Encrypt免费证书
1. 网站详情 → 「SSL」选项卡
2. 先注册ACME账号（填写邮箱）
3. 选择Let's Encrypt证书
4. 填写需要签发的域名
5. 选择验证方式（HTTP验证/DNS验证）
6. 点击申请，系统自动完成验证并签发
7. 证书到期前可续签

### 自签名证书
1. 网站详情 → 「SSL」选项卡
2. 选择「自签名证书」
3. 首次使用需先创建根证书
4. 使用根证书签发站点证书
5. 可下载根证书PFX文件导入浏览器信任

### 证书管理
- 支持上传第三方证书
- 支持证书到期提醒
- 支持证书自动续签
- 支持证书下载备份

### 开启HTTPS
- 部署SSL证书后自动开启HTTPS
- 证书信息显示到期时间和域名列表
- 支持HTTP自动跳转HTTPS
- 支持HSTS配置

## 反向代理

### 创建反向代理
1. 网站详情 → 「反向代理」
2. 点击「添加反向代理」
3. 填写代理名称
4. 填写目标URL（如 http://127.0.0.1:8080）
5. 配置代理选项
6. 保存

### 代理配置
- 支持HTTP/HTTPS/WebSocket代理
- 支持自定义请求头
- 支持代理缓冲配置
- 支持代理超时设置

### 负载均衡
- 支持多后端服务器负载均衡
- 支持轮询、最少连接、IP Hash算法
- 支持后端健康检查
- 支持故障转移

## 网站操作

- **启动/停止**：控制网站运行状态
- **网站目录**：快速跳转到文件管理器
- **Nginx配置**：查看和编辑站点Nginx配置文件
- **访问日志/错误日志**：可单独开关
- **网站到期设置**：可设置到期时间，到期后自动停止

## 网站分组

- 支持网站分组管理
- 创建自定义分组
- 按分组筛选网站
- 批量操作分组内网站

## 伪静态

### 伪静态规则
- 网站详情 → 「伪静态」
- 支持自定义Rewrite规则
- 内置常用框架规则（WordPress、ThinkPHP等）
- 支持正则表达式

### 常用规则
- WordPress伪静态
- ThinkPHP伪静态
- Laravel伪静态
- Django URL重写

## 防盗链

### 防盗链配置
- 网站详情 → 「防盗链」
- 配置允许的引用来源
- 配置禁止的文件类型
- 支持白名单域名

## 网站备份

- 网站详情 → 「备份」→「创建备份」
- 备份文件存储在备份目录中
- 支持从备份恢复网站
- 支持备份到云存储

## 访问控制

### IP访问控制
- 支持IP白名单访问
- 支持IP黑名单禁止
- 支持IP段配置

### 用户认证
- 支持HTTP Basic认证
- 配置用户名和密码
- 适用于后台管理页面保护

## 使用场景

### 企业官网
- 创建静态网站或PHP网站
- 配置SSL证书实现HTTPS
- 绑定多个域名

### Web应用
- 部署Python/Go/Node.js应用
- 配置反向代理
- 配置负载均衡

### API服务
- 创建API站点
- 配置反向代理到后端服务
- 配置CORS跨域

## 注意事项

- 域名需先解析到服务器IP
- SSL证书申请需要域名解析正确
- 反向代理目标服务需正常运行
- 修改Nginx配置后需重载生效
- 网站目录权限需正确设置
