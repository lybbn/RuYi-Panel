# 网站管理

## 创建网站

如意面板支持以下网站类型：

### 静态网站
1. 进入「网站」页面，点击「创建网站」
2. 选择「静态网站」类型
3. 填写域名（支持泛域名 *.example.com），可绑定多个域名
4. 设置根目录路径
5. 选择站点分组
6. 点击提交，系统自动生成Nginx配置并创建站点

### Python项目
1. 在「Python」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径和启动方式
4. 选择Python版本（需先在应用商店安装对应版本）
5. 配置框架类型（Django/Flask等）和启动模块
6. 系统自动创建站点并配置Nginx反向代理

### PHP项目
1. 在「PHP」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径
4. 选择PHP版本（需先在应用商店安装对应版本）
5. 系统自动创建站点并配置PHP-FPM

### Go项目
1. 在「Go」管理页面，点击「创建项目」
2. 填写项目名称和域名
3. 设置项目路径和启动命令
4. 选择Go版本（需先在应用商店安装对应版本）
5. 系统自动创建站点并配置Nginx反向代理

## 域名管理

- 每个网站支持绑定多个域名（含端口）
- 网站详情 → 「域名管理」→「添加域名」
- 域名格式：domain.com 或 domain.com:8080
- 域名变更后自动更新Nginx配置

## SSL证书

### Let's Encrypt免费证书
1. 网站详情 → 「SSL」选项卡
2. 先注册ACME账号（填写邮箱）
3. 选择Let's Encrypt证书
4. 填写需要签发的域名
5. 选择验证方式（HTTP验证/DNS验证）
6. 点击申请，系统自动完成验证并签发
7. 证书到期前可续签

### 自签名证书
1. 网站详情 → 「SSL」选项卡
2. 选择「自签名证书」
3. 首次使用需先创建根证书
4. 使用根证书签发站点证书
5. 可下载根证书PFX文件导入浏览器信任

### 开启HTTPS
- 部署SSL证书后自动开启HTTPS
- 证书信息显示到期时间和域名列表

## 网站操作

- **启动/停止**：控制网站运行状态
- **网站目录**：快速跳转到文件管理器
- **Nginx配置**：查看和编辑站点Nginx配置文件
- **访问日志/错误日志**：可单独开关
- **网站到期设置**：可设置到期时间，到期后自动停止

## 网站备份

- 网站详情 → 「备份」→「创建备份」
- 备份文件存储在备份目录中
- 支持从备份恢复网站
