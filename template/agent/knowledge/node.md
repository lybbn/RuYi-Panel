# 节点管理

tags: 节点,集群,负载均衡,多服务器,文件传输,数据库同步,监控

## 节点管理概述

如意面板支持多节点管理，可统一管理多台服务器，实现集群管理、负载均衡、文件传输等功能。

## 节点类型

### API节点
- 通过API密钥连接远程面板
- 需要远程服务器也安装如意面板
- 支持完整功能管理

### SSH节点
- 通过SSH协议连接远程服务器
- 不需要远程安装面板
- 支持基础运维操作

## 添加节点

### 添加API节点
1. 「节点」→「节点列表」页面
2. 点击「添加节点」
3. 选择节点类型为「API节点」
4. 填写节点名称和地址（如 http://remote-ip:6789）
5. 填写API Key
6. 选择节点分类
7. 添加备注
8. 保存

### 添加SSH节点
1. 「节点」→「节点列表」页面
2. 点击「添加节点」
3. 选择节点类型为「SSH节点」
4. 填写节点名称和服务器IP
5. 填写SSH端口（默认22）
6. 填写用户名
7. 选择认证方式（密码/密钥）
8. 填写密码或上传密钥
9. 保存

## 节点管理

### 节点信息
- 节点名称、地址、类型
- 节点状态（正常/连接失败/未连接）
- 操作系统信息
- CPU、内存、磁盘使用率
- 运行时间
- 最后监控时间

### 节点操作
- **连接测试**：测试节点连接状态
- **远程终端**：打开远程服务器终端
- **重启服务器**：远程重启服务器
- **重启面板**：远程重启面板服务
- **编辑节点**：修改节点配置
- **删除节点**：移除节点

### 节点分类
- 支持节点分组管理
- 创建自定义分类
- 按分类筛选节点

## 负载均衡

### Upstream资源组
- 「节点」→「负载均衡」页面
- 创建Upstream资源组
- 配置后端服务器列表
- 设置负载均衡算法

### 负载均衡算法
- **轮询（round_robin）**：依次分配请求
- **最少连接（least_conn）**：分配到连接数最少的服务器
- **IP Hash（ip_hash）**：同一IP始终分配到同一服务器

### 协议支持
- HTTP负载均衡
- TCP负载均衡
- UDP负载均衡

### 配置选项
- 连接超时设置
- 读取超时设置
- 发送超时设置
- 保持连接数
- 故障转移策略

## 文件传输

### 传输功能
- 「节点」→「文件传输」页面
- 在节点之间传输文件
- 支持上传到远程节点
- 支持从远程节点下载

### 传输操作
1. 选择源节点和目标节点
2. 选择源文件路径
3. 选择目标路径
4. 开始传输
5. 查看传输进度

## 注意事项

- API Key请妥善保管
- SSH连接建议使用密钥认证
- 负载均衡需配合Nginx使用
- 数据库同步需配置server-id
- 节点间网络延迟会影响同步性能
