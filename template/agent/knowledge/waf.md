# WAF防火墙

tags: WAF,防火墙,安全,攻击防护,CC防护,SQL注入,XSS,规则

## WAF概述

如意面板内置WAF（Web应用防火墙），保护网站免受常见Web攻击。每个网站可独立开启或关闭WAF防护。

## 站点防护

### 开启WAF
1. 「WAF」页面 → 站点列表
2. 选择需要防护的网站
3. 开启WAF防护开关
4. 系统自动应用默认防护规则

### 防护模式
- **观察模式**：仅记录不拦截，用于测试规则
- **拦截模式**：记录并拦截攻击请求

## 防护规则

### 内置规则
- SQL注入防护
- XSS跨站脚本防护
- 命令注入防护
- 文件包含防护
- 扫描器识别
- 常见漏洞利用防护

### 自定义规则
1. 「WAF」→「规则管理」
2. 点击「添加规则」
3. 设置匹配条件：URI / 参数 / Header / Body
4. 选择匹配方式：包含 / 正则 / 等于
5. 设置动作：拦截 / 放行 / 验证码

## CC防护

### CC攻击防护
- 自动识别异常高频请求
- 可设置单IP请求频率阈值
- 超出阈值自动触发拦截

## 访问统计

### 攻击日志
- 「WAF」→「攻击日志」查看拦截记录
- 显示攻击类型、来源IP、请求URL、时间
- 支持按类型、IP、时间筛选

## 白名单

### IP白名单
- 添加信任IP，跳过WAF检测
- 适用于CDN节点、搜索引擎爬虫

### URL白名单
- 指定URL路径跳过WAF检测
- 适用于API接口、回调地址
