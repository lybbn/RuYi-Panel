# WAF防火墙

tags: WAF,防火墙,安全,攻击防护,CC防护,SQL注入,XSS,规则,IP黑名单,IP白名单,UA名单,地域限制,报表

## WAF概述

如意面板内置WAF（Web应用防火墙），保护网站免受常见Web攻击。每个网站可独立开启或关闭WAF防护。

## 全局配置

### WAF开关
- 「WAF」→「全局配置」页面
- 三种模式：关闭、观察模式（仅记录）、防护模式（拦截）
- 全局开关控制所有站点的WAF状态

### 基础配置
- 日志保留天数配置
- 请求体大小限制
- 自定义错误页面

## 站点防护

### 开启WAF
1. 「WAF」页面 → 站点列表
2. 选择需要防护的网站
3. 开启WAF防护开关
4. 系统自动应用默认防护规则

### 防护模式
- **观察模式**：仅记录不拦截，用于测试规则
- **拦截模式**：记录并拦截攻击请求

### 站点独立配置
- 每个站点可独立配置防护规则
- 支持站点级白名单设置
- 支持站点级CC防护配置

## 防护规则

### 内置规则
- SQL注入防护
- XSS跨站脚本防护
- 命令注入防护
- 文件包含防护
- 扫描器识别
- 常见漏洞利用防护
- WebShell检测
- 敏感信息泄露防护

### 规则分类
- OWASP CRS规则集
- 自定义规则集
- 规则按严重级别分类（低/中/高/严重）

### 自定义规则
1. 「WAF」→「规则管理」
2. 点击「添加规则」
3. 设置匹配条件：URI / 参数 / Header / Body
4. 选择匹配方式：包含 / 正则 / 等于
5. 设置动作：拦截 / 放行 / 验证码
6. 可设置规则优先级

## CC防护

### CC攻击防护
- 自动识别异常高频请求
- 可设置单IP请求频率阈值
- 超出阈值自动触发拦截
- 支持设置CC防护时间窗口
- 支持设置CC防护触发次数

### CC防护策略
- **IP频率限制**：限制单IP请求频率
- **URI频率限制**：限制单URI访问频率
- **人机验证**：触发后要求输入验证码

## IP管理

### IP黑名单
- 「WAF」→「IP名单」→「黑名单」
- 添加恶意IP地址或IP段（支持CIDR）
- 黑名单IP直接拦截访问
- 支持批量导入IP
- 支持设置封禁时长

### IP白名单
- 「WAF」→「IP名单」→「白名单」
- 添加信任IP，跳过WAF检测
- 适用于CDN节点、搜索引擎爬虫、合作伙伴
- 支持IP段白名单

### IP自动封禁
- 攻击次数超过阈值自动加入黑名单
- 可配置自动封禁策略
- 支持自动解封时间

## UA名单

### UA黑名单
- 「WAF」→「UA名单」→「黑名单」
- 封禁恶意User-Agent
- 支持正则匹配
- 常见封禁：扫描器、爬虫、恶意软件

### UA白名单
- 「WAF」→「UA名单」→「白名单」
- 放行特定User-Agent
- 适用于搜索引擎爬虫、监控工具

## 地域限制

### 地域封禁
- 「WAF」→「地域限制」
- 按国家/地区限制访问
- 适用于仅面向特定地区用户的服务
- 使用GeoIP数据库识别

### 地域放行
- 仅允许特定地区访问
- 其他地区自动拦截

## URL白名单

### URL白名单设置
- 指定URL路径跳过WAF检测
- 适用于API接口、回调地址、Webhook
- 支持精确匹配和前缀匹配

## 访问统计

### 攻击日志
- 「WAF」→「攻击日志」查看拦截记录
- 显示攻击类型、来源IP、请求URL、时间
- 支持按类型、IP、时间筛选
- 支持查看完整请求详情
- 支持导出日志

### 仪表盘
- 「WAF」→「仪表盘」页面
- 攻击趋势图（24小时/7天/30天）
- 攻击类型分布饼图
- Top攻击IP排行
- Top攻击URL排行
- 拦截率统计

### 报表统计
- 支持按时间范围生成报表
- 攻击类型统计
- 攻击来源分析
- 站点防护统计
- 支持报表导出

## 规则更新

- WAF规则支持在线更新
- 定期更新防护规则库
- 支持手动导入规则

## 性能优化

- WAF规则缓存优化
- 支持规则预编译
- 低延迟高性能设计
- 支持白名单快速放行
