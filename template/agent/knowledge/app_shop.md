# 应用商店

tags: 应用商店,软件安装,Nginx,MySQL,PHP,Redis,Docker,Python,Go,Supervisor,Fail2Ban,安装,卸载

## 安装应用

1. 进入「应用商店」页面
2. 浏览或搜索需要的应用
3. 点击「安装」按钮，选择安装版本
4. 确认安装，系统自动下载并配置
5. 安装进度在「任务管理器」中查看

## 应用分类

### Web服务器
- **Nginx**：轻量级Web服务器，支持OpenResty版本（推荐）和原生Nginx版本

### 数据库
- **MySQL**：5.7和8.0版本可选
- **Redis**：3.0和5.0版本（Windows），7.2和7.4版本（Linux）

### 运行环境
- **Docker**：Linux支持27.5/28.0/29.0版本，Windows为Docker Desktop
- **Python**：支持3.12、3.13、3.14多个版本，可同时安装
- **Go**：支持1.23和1.26版本，可同时安装
- **PHP**：支持7.4、8.1、8.2、8.3、8.4版本，可同时安装

### 安全防护
- **Supervisor**：进程守护管理工具，可重启崩溃或退出的进程和设置开机启动
- **Fail2Ban**：防暴力破解服务，保护SSH、Web等服务（仅Linux）

## 应用管理

### 启停操作
- 应用列表中可对已安装应用执行启动/停止/重启/重载
- Nginx支持重载配置（reload）无需重启

### 卸载应用
1. 应用列表 → 点击「卸载」
2. Python/Go/PHP支持多版本共存，卸载时需指定版本
3. MySQL卸载前需先删除所有关联数据库

### 配置管理
- 每个应用支持查看和编辑配置文件
- 修改配置后自动重载服务生效
- Nginx和MySQL支持性能参数调整

### 日志查看
- 支持查看错误日志、访问日志、慢查询日志
- 默认显示最近2000行

## PHP专项管理

- PHP扩展管理：安装/卸载扩展
- PHP-FPM配置：进程池参数调整
- PHP配置：禁用函数管理、配置参数调整
- 支持查看phpinfo信息
- 支持PECL扩展安装
- 支持OPcache清理
